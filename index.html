The h keyword is used to show things in a bold manner as a heading.

Headings of same name can be created without any problem

The p keyword is used to show things as a paragraph in a non bold manner

The hr keyword is used to create a horizontal line in the webpage


The br keyword is used to start the heading of paragraph from the next line

The q keyword is used to quote a particular part in a paragraph

The blockquote keyword leaves a gap before starting the sentence
